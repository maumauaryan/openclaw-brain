## The Content Funnel

### 🔴 Top of Funnel — Awareness
**Goal:** Get Eyeballs
- Short-form viral videos
- Educational, broad topics
- 5–10x average views rule
- Never try to sell in short-form

---

### 🟡 Middle of Funnel — Trust & Authority
**Goal:** Build Trust
- Long-form YouTube (30-60 min value bombs)
- Strategy breakdowns
- Client onboardings live
- In-person event recordings

---

### 🟢 Bottom of Funnel — Conversions
**Goal:** Generate Sales
- Instagram Story sequences (client results, overwhelm with proof)
- Client interviews (podcast-style)
- Social proof overload

## The Hook Library

### 6 Viral Hook Archetypes

**1. Mute Hooks (Most Powerful)**
- Videos that need no language
- Visual storytelling only
- Best for: Going global, maximum reach

**2. Strong Polarizing Takes**
- "What used to work on IG is dead"
- Triggers emotion, creates curiosity

**3. Immediately Applicable Quick Tips**
- "These 3 Instagram settings get you more views"
- High saves, easy to create

**4. Starting From Scratch**
- "If I had to start my content from zero"
- Beginner magnet, high relatability

**5. Reactions & Breakdowns**
- React to Alex Hormozi clip
- Borrow authority + pre-existing virality

**6. Split Screen / Duality**
- "100 views video vs 1M views video" (40M views)
- Combine with mute

## Scripting System

### The 5-Part CCC Viral Script Structure

1. **HOOK** — Capture attention in 0–3 seconds
2. **SELLING THE SOLUTION** — Create desire (ZERO value given here)
3. **GIVING THE PRINCIPLE** — Explain the concept
4. **MAKING IT APPLICABLE** — Give practical steps
5. **CALL TO ACTION** — One action only

### Script Writing Golden Rules

**⚡ RULE 1** — Keep it 6th Grade Level
- Use readability checker
- 5th–6th grade target
- Simpler = master

**✓ RULE 2** — Value Per Minute
- Cross out unnecessary sentences
- Short-form is about DENSITY
- More value per second = more rewatches

## Content Ideation

### The Viral Discovery Method

**STEP 1** — Strategic Search
- Find accounts in niche (50K–500K followers)
- Look for videos with 10K+ likes
- Go to Reels tab

**STEP 2** — Outlier Identification
- Calculate average views (last 10–15 posts)
- Find 5–10x MORE than average
- SortFeed extension speeds this up

**STEP 3** — Hook Deconstruction
- Write down: Spoken hook, Visual hook, Text hook
- Store in Notion database
- Patterns emerge over time

**STEP 4** — Ethical Adaptation
- Keep exact hook structure and format
- Change value completely (your expertise)
- Not copying — using proven packaging
